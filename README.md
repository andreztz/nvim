## Check Healthy

Ao iniciar pela primeira vez digite:

```
:checkhealth
```

Alguns avisos podem aparecer, é necessário corrigi-los:


```
$ sudo pacman -S prettier stylua xsel
```

Suporte python

```
$ pip install pynvim
```


Para instalar um servidor de liguagem digite:

```
:LspInstallInfo
```
